# 🌍 AI Travel Guide

It is a modern, responsive AI-powered travel guide website that helps users explore beautiful travel destinations through an interactive interface. Users can search destinations, filter by interests, save favorite places, and interact with an AI travel assistant for personalized recommendations.

## ✨ Features

- 🌎 Beautiful travel destination gallery
- 🔍 Search destinations by name or keyword
- 🏷️ Category-based filtering
- ❤️ Save favorite destinations
- 💬 AI Travel Guide chatbot
- 📱 Fully responsive design
- 🎨 Modern and clean user interface

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Akshaya-elumalai07/AI-Travel-guide.git
```

### Open the project

Simply open the `index.html` file in your browser.

Or use VS Code Live Server:

1. Open the project in VS Code.
2. Install the Live Server extension.
3. Right-click `index.html`.
4. Click **Open with Live Server**.
